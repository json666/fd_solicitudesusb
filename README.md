### Usage

``` bash
# clone the repo
$ git clone https://github.com/json666/fd_solicitudesusb.git

# go into app's directory
$ cd fd_solicitudesusb

# install dependencies
npm install

# serve with hot reload at localhost:4200
ng serve

# build for production with minification
ng build
```
